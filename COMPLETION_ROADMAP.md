## 🔥 남은 작업 (5%)

### 우선순위 1: Sepolia 테스트넷 실제 배포 (필수)
**예상 소요 시간**: 15분  
**복잡도**: ⭐⭐ (낮음)

**작업 항목**
1. 환경변수 설정 (PRIVATE_KEY, SEPOLIA_RPC_URL, ETHERSCAN_API_KEY)
2. `npx hardhat run scripts/deploy.ts --network sepolia` 실행
3. 배포된 컨트랙트 주소 저장
4. Etherscan 검증 및 ABI 확인

**기대 효과**
- 온체인 추첨 결과 기록 가능
- 당첨자 검증 자동화
- 블록체인 투명성 확보

---

### 우선순위 2: E2E 통합 테스트 (권장)
**예상 소요 시간**: 30분  
**복잡도**: ⭐⭐⭐ (중간)

**테스트 시나리오**
1. 무기명 로그인 → 인증코드 입력 → 자동 로그인
2. 구매 페이지 → 지갑 연동 → 월드코인 결제
3. Mempool 입금 감지 → 응모권 자동 발급
4. 관리자 로그인 → 당첨번호 입력 → 추첨 실행
5. 당첨자 선정 → 블록체인 기록 → 상금 청구

**테스트 도구**
- Vitest (단위 테스트)
- Postman (API 테스트)
- 수동 테스트 (UI/UX)
